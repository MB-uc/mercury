# Document checklist — Mercury intelligence skills

**Version**: 1.0 (rebuilt)
**Date**: 17 February 2026
**Total items**: 130

## How to read this checklist

Each item has three tags:

**Skill assignment**
- **CR** — company-research owns this item (found via external sources, filings, search)
- **WR** — website-research owns this item (found on the company's own website)
- **BOTH** — both skills check for it from their respective angles

**Automation tier**
- **T1** — fully automatable now via web scraping, search, and free APIs
- **T2** — partially automatable; the document can be found but full extraction or analysis is limited
- **T3** — not yet available; requires paid subscriptions, manual input, or specialist tools

**How skills use it**
- company-research works through CR-assigned items during its research phases and explicitly flags T3 items as gaps with suggested sources
- website-research checks WR-assigned items against what it finds on the site, marking each as: present (with URL), absent, partial, gated, or external
